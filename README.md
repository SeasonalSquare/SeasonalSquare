- 배포 URL: [http://j3a503.p.ssafy.io/](http://j3a503.p.ssafy.io/)

## :scroll: 기능
- 가격, 제철 시기, 사용자의 알레르기, 비건 정보를 고려한 농수산물 추천
- 추천된 농수산물을 활용한 레시피 추천
- 레시피 재료 체크를 위한 장바구니 기능
- 레시피의 예상 가격 계산
